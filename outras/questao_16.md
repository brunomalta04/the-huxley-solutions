# Questão 16 - Bingo
## Descrição
Você está organizando um bingo e deseja que cada número sorteado apareça no telão.

## Formato de entrada
- Um número inteiro.

## Formato de saída
- A mensagem: 
  \\\" >> questao_16.md
echo 
O
numero
sorteado
foi:
X >> questao_16.md
echo 
\\\"
  Onde **X** é o número digitado.

## Exemplo de entrada e saída:

**Entrada:**
\\\" >> questao_16.md
echo 
3157 >> questao_16.md
echo \\\"

**Saída:**
\\\" >> questao_16.md
echo 
O
numero
sorteado
foi:
3157 >> questao_16.md
echo \\\"
